## Physical Data

Fixture visualization requires Physical data to be correctly entered into the
GDTF file. If this data is incorrect or missing the fixture will not visualize
correctly.

If the fixture is missing physical data, Blender provides default physical
data. This data can be enabled by pressing the "Do Not Use Fixtures Physical
Properties". Fixture data can be enabled by pressing the "Use Fixtures Physical
Properties". 

### Default Physical Data provided by BlenderDMX:

- Dimmer: 0 to 1
- Zoom: 0 to 120°
- Playmode: 0 to 0
- Recording: 0 to 0
- Pan: 270  to -270°
- Tilt: -135 to 135°
- PanRotate: 500 to -500°/s
- TiltRotate: 500 to -500°/s
- Gobo1Pos: -180 to 180°
- Gobo1PosRotate: -1000 to 1000°/s
- Gobo2Pos: -180 to 180°
- Gobo2PosRotate: -1000 to 1000°/s
- CTC: 2700 to 12000K
- CTB: 2700 to 12000K
- CT0: 2700 to 12000K
- Iris: 0 to 1
- Shutter1: 0 to 1
- Shutter1Strobe: 0.3 to 20Hz
